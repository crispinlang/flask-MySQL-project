FAQ

Frequently Asked Questions

What is this website for?

This site showcases results from a MySQL database using a Flask backend. It's mainly for experimentation and learning, and for visualizing data from a previous project.

How are the results loaded?

The backend uses Python and Flask to connect to a MySQL database, run SQL queries, and pass the results into Jinja templates which render the tables you see.

Can I see multiple tables?

Yes. The results and testing pages demonstrate loading one or several tables and rendering them in separate styled boxes.
